解决virtualbox设置VM桥接模式没有ip问题

Posted on 2018-08-31

启动虚拟机没有为什么IP？

  • DHCP需要一点时间的，centos7默认会配置ip的，需要30秒左右的等待时间

手动设置

  1. 设置ip
  • ip addr add 10.100.100.7/24 dev eth0

    如果没有ip命令，先用NAT网络模式联网，然后安装 yum install net-tools

  1. 设置路由网关
  • ip route add default via 10.100.100.254 dev eth0

    如果不知道路由可以先在本地物理机查看

  1. 设置DNS
  • vi /etc/resolv.conf, 添加 nameserver 114.114.114.114
